# Description des classes
Dans l'ensemble de données Iris, on a trois classes qui représentent trois espèces d'iris.
Il y a Iris-setosa, Iris-versicolor et Iris-virginica, mais elles ont été changé par les 
classes 0, 1 et 2.

Dans l'ensemble de données Wine, il faut prédire si le vin est bon ou mauvais.
On a donc 2 classes, mauvais (0) et bon (1).
La classe 0 est équivalent à une qualité inférieur ou égale à 5 selon le jeu de donnée original.
La classe 1 est équivalent à une qualité supérieur ou égale à 7 selon le jeu de donnée original.

Dans l'ensemble de données Abalones, il y a trois classes qui représentent des intervalles
du nombre d'anneaux. 
La classe 0 représente le cas où il y a moins de 8 anneaux.
La classe 1 représente le cas où il y a entre 8 et 14 anneaux.
La classe 2 représente le cas où il y a plus de 14 anneaux.
